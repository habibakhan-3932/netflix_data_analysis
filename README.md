# netflix_data_analysis

Overview:
This project analyzes a Netflix dataset using Python, Pandas, and Matplotlib. 
The purpose is to clean the dataset, explore insights, and create visualizations.

Dataset Description:
The dataset contains:
- Title
- Type (Movie/TV Show)
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Date Added
- Description

Data Cleaning Performed:
1. Removed duplicate rows.
2. Handled missing values, especially in the date_added column.
3. Standardized date formats.
4. Cleaned duration field.
5. Fixed inconsistent text formatting.
6. Ensured proper data types for numeric and categorical columns.

Exploratory Data Analysis (EDA):
Visualizations created using Matplotlib:
- Movies vs TV Shows count (bar chart)
- Top 10 countries with most titles (bar chart)
- Content releases by year (line chart)
- Rating distribution (bar chart)
- Movie duration distribution (histogram)

Tools & Libraries Used:
- Python
- Pandas
- Matplotlib
- Jupyter Notebook or VS Code

Purpose of the Project:
This project demonstrates:
- How to clean a real dataset
- How to perform exploratory data analysis
- How to visualize insights using Python
- How to structure and present a GitHub-ready project

Final Notes:
This project is ideal for beginners building a data analysis portfolio.
It highlights skills in data cleaning, visualization, and documentation.
